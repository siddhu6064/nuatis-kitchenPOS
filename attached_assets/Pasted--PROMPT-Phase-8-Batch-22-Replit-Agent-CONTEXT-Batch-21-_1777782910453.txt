SCOPE — BUILD THIS
1. Migration 0017 — order_discounts table:
   - id (uuid pk), tenant_id (uuid, FK tenants, RLS), order_id (uuid, FK orders)
   - type ('pct' | 'amt') as text with check constraint
   - value_bps int null (used when type='pct'; bps so 1000 = 10%)

   - applied_amount_cents int not null (the realized cents subtracted at apply time)

2. Endpoint POST /v1/orders/:id/discount — manager PIN gated. Body: { type, value, reason }.
   - Validates: value > 0; pct value_bps ≤ 5000 (50% cap); amt value_cents ≤ current
     subtotal_cents; reason 1-200 chars
   - Inserts order_discounts row
   - Calls recalcOrderTotals() — now factors in non-voided discounts
   - Returns updated order with discount line in response
   - tenant_id from req.auth, never body
   - 400 on validation failure; 409 if order is paid/voided

3. Endpoint POST /v1/orders/:id/discount/:application_id/void — manager PIN gated.
   Sets voided_at + voided_by_staff_id, recomputes totals. POST not DELETE so it shows
   up in audit trail as an action.

4. Recompute logic in lib/db.ts (extend existing recalcOrderTotals):
   - subtotal_cents = sum of order_items lines — UNCHANGED
   - Apply non-voided discounts in applied_at order:
     - if pct: this_discount = floor(subtotal_cents * value_bps / 10000)
     - if amt: this_discount = min(value_cents, max(0, subtotal_cents - sum_of_prior_applied))
     - persist applied_amount_cents on the row
   - discount_total_cents = sum of applied_amount_cents (non-voided)
   - taxable_subtotal_cents = sum of items where taxable=true (UNCHANGED)
   - non_taxable_subtotal_cents = subtotal_cents - taxable_subtotal_cents
   - Proportional tax reduction:
     taxable_after_discount = floor(
       taxable_subtotal_cents * (subtotal_cents - discount_total_cents) / max(subtotal_cents, 1)
     )
   - tax_cents = floor(taxable_after_discount * sales_tax_bps / 10000)
   - total_cents = subtotal_cents - discount_total_cents + tax_cents + tip_cents
   - Persist discount_total_cents on orders row (denormalized for read speed)
   - All math in cents int; use Math.floor() consistently; never round
